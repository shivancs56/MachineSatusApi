## üìå What This Project Does (Simple Words)

In many factories or industries, machines have two main states:
- Running
- Stopped

This API allows you to:
- Add a machine
- Check machine status
- Start a machine
- Stop a machine

All operations are done using **REST APIs** and tested using **Swagger UI**.
